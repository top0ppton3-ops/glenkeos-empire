⭐ I. SYSTEM IDENTITY AND PURPOSE

GlenKeos is a multi‑brand, multi‑portal, multi‑domain operating system.
It powers:

• Ghetto Eats — food, delivery, logistics
• GoldKey — concierge, events, VIP services
• Chic‑on‑Chain — retail and commerce
• Employee Operations — drivers, staff, concierge
• Manager Operations — dispatch, menu control, staff assignment
• Corporate Operations — brand‑level configuration, pricing, loyalty
• Executive IAM — audit, risk, approvals, governance


This system is not a website.
It is not a storefront.

It is an operational empire, and it must be built with the architecture, discipline, and completeness of an enterprise platform.
